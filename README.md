genpwd
======

`genpwd` is a simple and pure-Python password generator.  It generates
variable-length, secure passwords by making use of `os.urandom()`.

Running `genpwd`
----------------

To generate a password, simply run:

    $ ./genpwd.py

To change the password length, run:

    $ ./genpwd.py -l 40
